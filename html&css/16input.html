$("[readonly]").click( function(){ console.log("clicked") });//log - clicked $("[readonly]").hover( function(){ console.log("hovered") });//log - hovered $("[readonly]").dblclick( function(){ console.log("double clicked") });//log - double clicked

None of the below will get triggered. $("[disabled]").click( function(){ console.log("clicked") });//No Impact $("[disabled]").hover( function(){ console.log("hovered") });//No Impact $("[disabled]").dblclick( function(){ console.log("double clicked") });//No Impact

相同点：都会使表单变成不可编辑的状态

不同点：

  1. readonly是只读（可能是某种处理），disabled是禁用（可能是权限或其他原因），意义不同
  2. disabled会使文本框变灰，readonly不会
  3. disabled对所有的表单都能生效，readonly对hidden、radio、checkbox、range、color无效
  4. disabled的值不会被form收集，readonly的值会。
  5. readonly只对input的text属性和textarea有效。
